Question-1: What are the different ways to select an element in the DOM?

There are some ways to select an element in the DOM.
There are few:
i. getElementById() ----- to select an element using its id
ii. getElementsByClassName() ----- to select an or multiple item with the same class
iii. getElementsByName() ---- to select an or collection of elements using name attribute
iv. querySelectorAll() ---- returns all descendent selectors from the specific selector
v. querySelector() ---- returns the first descendent child

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML ---- it gets or set the inner HTML elements within the element. It contains tags,attribute,content etc.
innerText ---- It gets or sets the text of an element. It can only show text which are rendered in screen but can not access text which are hidden using CSS or other methods.
innerContent ---- It also gets or sets the text of an element. It can access both text which are rendered on display and hidden using CSS or other method. It do not access any styling of text and return only raw text.

Question-3: What is event delegation in the DOM?

It is an event handling pattern that allows us to control an event at a higher level rather than the level where the event was first happened. The idea is to delegate the event to the parent element and control its root element where the event happens.
For example: if we have 3 buttons in a div. We can access events in button using div. [event.target.tagName]

Question-4: What is event bubbling in the DOM?

It happens when an element receives an event, that event bubbles up or transmitted to its parent elements in the DOM until it reach to the root element.
Example: section>div>button -- here the child element is button. If the button receives the click event it will pass that event to div, then the event also passes to the section until ite reach HTML element which is the root element in html.
